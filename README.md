# RJYA
Russian Sign Language learning app
Приложение по изучения РЖЯ(русский жестовый язык) на русском языке. An application for learning Russian Sign Language (RSL) in Russian.
С интерактивными заданиями. With interactive tasks.
Шаг 1. Запуск приложения. Действие открыть приложение через значок на главном экране. Ожидаемый результат отображается главный экран с кнопками:
–«Информация»;
–«…Урок…»;
–«Упражнения»;
–«Слова на дактиле»;
–«О приложении».
Step 1. Launch the app. Open the app via the icon on the home screen. The expected result is the home screen with the following buttons:
– "Information";
– "...Lesson...";
– "Exercises";
– "Dactyl Words";
– "About the App."

Шаг 2. Изучение карточек. Действие. нажать «…Урок…», выбрать тему «Алфавит (Дактиль)» Рисунок Г.2, открыть карточку с буквой «А». Ожидаемый результат:
–отображается изображение жеста буквы «А». ;
–есть название жеста;
–присутствует видео;
–доступна кнопка замедления воспроизведения;
–Работают кнопки «назад».
Step 2. Studying the Flashcards. Action: Click "...Lesson...", select the topic "Alphabet (Dactyl)" (Figure D.2), and open the flashcard with the letter "A". Expected result:
– the image of the letter "A" gesture is displayed.;
– the gesture name is present;
– the video is present;
– the slow-motion button is available;
– the back buttons are functional.

Упражнения. Ожидаемый результат:
–загружается видеовопрос «Какой жест показан?»;
–четыре варианта ответа;
–после выбора — переход к следующему вопросу;
–по завершении — сообщение «Тест завершен».
Exercises. Expected result:
– the video question "What gesture is shown?" is loaded ;
– four answer options;
– after selecting, move on to the next question;
– upon completion, the message "Test completed."

Шаг 5. Проверка раздела «О приложении» Действие. Нажать «О приложении». Ожидаемый результат:
–отображается информация о версии, разработчике;
–доступна контактная информация для обратной связи.
Step 5. Checking the "About the app" section. Action: Click "About the app." Expected result:
–version and developer information are displayed;
–contact information is available for feedback.
<img width="463" height="644" alt="image" src="https://github.com/user-attachments/assets/d2d152bc-2d9a-42b7-9e76-8a089a5109f6" />
Шаг 6. Технология дактиль. Действие. Вернуться на главный экран, нажать «Слово на дактиле». Ожидаемый результат:
–загружается форма для написания слова;
–записываешь любое слово нажимаешь проиграть;
–слово выводится по буквам;
–проигрывается и показывает каждую букву отдельно.
Step 6. Dactyl Technology. Action. Return to the main screen and tap "Dactyl Word". Expected result:
– the word entry form loads;
– you type any word and tap "play";
– the word is displayed letter by letter;
– the game plays and displays each letter individually.
