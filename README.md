# RJYA
Russian Sign Language learning app
Приложение по изучения РЖЯ(русский жестовый язык) на русском языке. An application for learning Russian Sign Language (RSL) in Russian.
С интерактивными заданиями. With interactive tasks.
<br>Шаг 1. Запуск приложения. Действие открыть приложение через значок на главном экране. Ожидаемый результат отображается главный экран с кнопками:
<br>–«Информация»;
<br>–«…Урок…»;
<br>–«Упражнения»;
<br>–«Слова на дактиле»;
<br>–«О приложении».
<br>Step 1. Launch the app. Open the app via the icon on the home screen. The expected result is the home screen with the following buttons:
<br>– "Information";
<br>– "...Lesson...";
<br>– "Exercises";
<br>– "Dactyl Words";
<br>– "About the App."

<br>Шаг 2. Изучение карточек. Действие. нажать «…Урок…», выбрать тему «Алфавит (Дактиль)», открыть карточку с буквой «А». Ожидаемый результат:
<br>–отображается изображение жеста буквы «А». ;
<br>–есть название жеста;
<br>–присутствует видео;
<br>–доступна кнопка замедления воспроизведения;
<br>–Работают кнопки «назад».
<br>Step 2. Studying the Flashcards. Action: Click "...Lesson...", select the topic "Alphabet (Dactyl)" (Figure D.2), and open the flashcard with the letter "A". Expected result:
<br>– the image of the letter "A" gesture is displayed.;
<br>– the gesture name is present;
<br>– the video is present;
<br>– the slow-motion button is available;
<br>– the back buttons are functional.

<br>Упражнения. Ожидаемый результат:
<br>–загружается видеовопрос «Какой жест показан?»;
<br>–четыре варианта ответа;
<br>–после выбора — переход к следующему вопросу;
<br>–по завершении — сообщение «Тест завершен».
<br>Exercises. Expected result:
<br>– the video question "What gesture is shown?" is loaded ;
<br>– four answer options;
<br>– after selecting, move on to the next question;
<br>– upon completion, the message "Test completed."

<br>Шаг 5. Проверка раздела «О приложении» Действие. Нажать «О приложении». Ожидаемый результат:
<br>–отображается информация о версии, разработчике;
<br>–доступна контактная информация для обратной связи.
<br>Step 5. Checking the "About the app" section. Action: Click "About the app." Expected result:
<br>–version and developer information are displayed;
<br>–contact information is available for feedback.

<br>Шаг 6. Технология дактиль. Действие. Вернуться на главный экран, нажать «Слово на дактиле». Ожидаемый результат:
<br>–загружается форма для написания слова;
<br>–записываешь любое слово нажимаешь проиграть;
<br>–слово выводится по буквам;
<br>–проигрывается и показывает каждую букву отдельно.
<br>Step 6. Dactyl Technology. Action. Return to the main screen and tap "Dactyl Word". Expected result:
<br>– the word entry form loads;
<br>– you type any word and tap "play";
<br>– the word is displayed letter by letter;
<br>– the game plays and displays each letter individually.
